# Automatic-Railway-Ticket-Booking-System-using-Python
Automatic Railway Ticket Booking System will enable you to automatically fill all the details so that it saves your time to book an Tatkal Ticket.

You have to set your time in the schedule function at that paticular time it will open an UI which will be autofill with your predefined booking details you have to click next and Pay Now in the UI for successfully completing the execution of program.  

It also creates an log file which will store the booking details with specific date an time as file name.
